# Kiddo concept references (design only)

Files in this folder are **approved KiddoLearn concept sheets for design reference**.

- Do **not** auto-crop these sheets for production use
- Do **not** treat SVG wireframes in `_archive-generated-svg/` as final art
- Export dedicated production PNGs into the library folders (e.g. `../characters/`)

## Production asset libraries

| Folder | Purpose |
|--------|---------|
| `../characters/` | Character portraits (Character Library — active) |
| `../branding/` | Logo, hero, splash (planned) |
| `../worlds/` | Learning world art (planned) |
| `../gameplay/` | App preview, quests, rewards (planned) |
| `../ui/` | Parent dashboard, badges, icons (planned) |

Asset paths are registered in `src/data/kiddoAssets.js`.
